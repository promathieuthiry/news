# News

An SPA displaying the latest news from the [News Api](https://newsapi.org/). 

## Table of contents

  - [General info](#general-info)
  - [Technologies](#technologies)
  - [Features](#features)

## General info

This project has no more ambitious than practice and having fun.

## Technologies

Project is created with:

* Create React App
* React Router
* A [suite](https://github.com/rsuite/rsuite) of React Components
* Dotenv to save variables
* Airbnb eslint configuration
* Deploy on netlify

## Features

* User can search for news by keyword or phrase
* User can navigate to the corresponding news article
* User can hide a news article post
* User can view the top news filter by country
* User can sort the top news by publication date
